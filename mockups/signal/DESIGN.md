# Signal

## Concept

Bold, modern, and energetic. Uses a vibrant indigo-to-cyan gradient as the signature element, chunky rounded cards with soft shadows, and strong visual hierarchy. The vibe is "forward-thinking engineer who builds the future."

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#ffffff` | Primary background |
| Alt Background | `#F9FAFB` | Alternating section backgrounds |
| Text Primary | `#111827` | Headings, names |
| Text Secondary | `#4B5563` | Body text, subtitles |
| Text Muted | `#9CA3AF` | Dates, metadata |
| Gradient Start | `#6366F1` | Indigo — gradient left/top |
| Gradient End | `#06B6D4` | Cyan — gradient right/bottom |

| Card Border | `#F3F4F6` | Subtle card borders |
| Pill BG (Indigo) | `#EEF2FF` | Data engineering / default skill pills |

| Pill BG (Amber) | `#FEF3C7` | AI/ML skill pills |
| Pill BG (Purple) | `#F3E8FF` | Language skill pills |

## Typography

- **All text**: Bold sans-serif stack (`Helvetica Neue`, `Helvetica`, `Arial`, `sans-serif`)
- **Name/hero**: Extra-bold (800 weight), large size, gradient fill
- **Section titles**: Bold (700), 28px
- **Card titles**: Bold (700), 18px
- **Body**: Regular weight, #4B5563
- **Labels/pills**: Small (11-13px), category-specific colors

## Layout

- Single column, max-width ~1100px, centered
- Alternating white and light gray section backgrounds
- Cards use 16px border-radius consistently
- 2-column grid for project cards, 3-column for skill cards
- Generous padding inside cards (24-28px)
- 80px+ spacing between sections

## Key Design Elements

- **Gradient is everywhere**: Hero name text, CTA buttons, section title underlines, card accent stripes, social icon backgrounds, contact section background
- **Rounded everything**: 16px border-radius on cards, pill-shaped nav links and skill tags, rounded CTA buttons
- **Gradient accent stripe**: Thin gradient bar at top of project and experience cards
- **Color-coded skill categories**: Each skill group has its own tinted pill color (indigo, green, amber, purple)
- **Pill-shaped navigation**: Nav links have rounded background pills
- **Social icons in gradient circles**: Filled gradient circles instead of outlined icons
- **Full-width gradient contact section**: Contact area breaks out of the content column with a gradient background and white CTA button (inverted contrast)
- **Soft shadows on hover** (implied — cards have subtle borders that would gain shadow on interaction)

## Navigation

- Frosted glass effect (semi-transparent white with blur)
- Bold gradient "JB" logo on the left
- Pill-shaped nav links with light gray background
- Circle theme toggle button on the right
- Thin bottom border

## Dark / Light Mode

- **Light-first design** — the bright, colorful palette is the primary aesthetic
- Dark mode: background to `#0F172A`, card backgrounds to `#1E293B`, text inverts
- The gradient accent is unchanged in both modes — it's the brand constant
- Dark mode makes the gradient elements pop even more against the dark surface
- Pill tag colors shift to slightly more saturated versions in dark mode

## Sections

- **Hero**: Gradient name text (largest element on page), "Hello, I'm" pre-label in accent color, location in a pill badge, social icons as gradient-filled circles, large gradient CTA button with rounded corners
- **About**: Light gray background section, gradient underline below title, body text paragraphs
- **Experience**: Stacked cards with gradient accent stripe on the left edge, role/company/date layout, description lines
- **Skills**: 3-column card grid, each card has a category label and color-coded rounded pills for each skill
- **Projects**: 2-column card grid, gradient accent stripe at top of each card, title, description, tech tags as small pills
- **Education**: Similar card format to experience
- **Contact**: Full-width gradient background breaking out of the content column, white heading, white inverted CTA button

## Mood / Inspiration

Communicates ambition, modernity, and technical sophistication. The gradient signals "I work at the cutting edge" — it's the visual language of modern tech products, AI tools, and developer platforms. The rounded, card-heavy layout feels approachable and organized. This design appeals to someone building the future — it's confident without being aggressive, colorful without being chaotic. It says "I take my craft seriously and I'm excited about what's next."
